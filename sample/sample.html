Sample Page Demo Script

Follow these steps to go through the sample demo. Make sure the servers are built and running, and that the browser extension is installed (see the README.md file for more information).

Pre-install a mDL

  1. Open the browser extension pop up window.
  2. Click on the "Config" tab.
  3. Type a "Domain", select the "mdl_1" schema, and import the demo mDL.

Get a JWT from the Issuer

  1. Go to the Issuer's page: http://contoso.com:8001.
  2. Login as Alice (username = "alice", password = "password").
  3. Click on the "Get JWT" button.
  4. Wait until the credential is ready.

Present the JWT

  1. Go to the Health Clinic's page: http://fabrikam.com:8004/login.
  2. Select and present the JWT in the Crescent browser extension.

Present the mDL

  1. Go to the Social page: http://soc.ial:8004/signup1.
  2. Select and present the mDL in the Crescent browser extension.
